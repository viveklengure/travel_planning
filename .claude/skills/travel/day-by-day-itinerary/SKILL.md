Generate a detailed day-by-day itinerary for: $ARGUMENTS

## Day-by-Day Itinerary

First, state the recommended trip duration and confirm it upfront. Then build the itinerary for that duration.

**Itinerary Principles**
- Group activities geographically to minimize backtracking
- Balance high-energy and relaxed activities within each day
- Include at least one meal recommendation per day tied to the itinerary location
- Allow buffer time — do not over-schedule
- Note when advance booking is required for an activity

---

For each day use the following structure:

### Day [N] — [Theme or Area Focus]

**Morning**
- Primary activity with location, estimated time needed, and why it's best done in the morning
- Optional add-on if time allows

**Afternoon**
- Primary activity with location, estimated time needed
- Lunch recommendation nearby (name, what to order)
- Optional add-on or alternative if energy allows

**Evening**
- Dinner recommendation (name, neighborhood, cuisine, price range)
- Evening activity or neighborhood to explore (sunset spot, night market, show, bar, etc.)
- Wind-down suggestion if appropriate

**Logistical Notes for the Day**
- How to get between locations (walk, metro, taxi, etc.)
- Anything that needs to be booked in advance for this day
- Estimated total spend for the day (mid-range traveler baseline)

---

After all days, include:

**Itinerary Flexibility Notes**
- Which days can be swapped or condensed for shorter trips
- What to cut first if the trip is shortened
- What to add first if the trip is extended by 1–2 days

**Pacing Advice**
- Flag any days that are particularly demanding
- Suggest where to build in a slow morning or an unscheduled afternoon
